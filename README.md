
## Tech Stack

- **Framework**: Static HTML
- **Styling**: Tailwind CSS (via CDN)
- **Typography**: Space Grotesk (body) + Inter (navigation)
- **Icons**: Custom SVG icons
- **Deployment Ready**: Works on any static host

## Features

- Fully responsive design
- Animated hero section with Orion constellation logo
- Scale.com-style dropdown navigation with glassmorphism panels
- Modern glassmorphism UI effects
- SEO-optimized meta tags
- Accessible markup
- Contact form with service selection
- Product pages with full details
- Blog with article pages
- Consistent constellation logo across all page footers

## Brand Identity

**Logo**: Orion constellation (5 stars connected by lines)

**Color Scheme**:
- Primary Blue: `#3b82f6` (Ark Solutions)
- Orange: `#ea580c` (Quantum Lab)
- Teal: `#0d9488` (Sprintsail)
- Purple: `#7c3aed` (App/Web Dev)
- Amber: `#d97706` (Shortlist)
- Dark backgrounds with glass effects

**Typography**:
- Body: Space Grotesk (weights 300-700)
- Navigation: Inter (weights 400-600)
- Headings: Bold, gradient text effects

## Local Development

Simply open any HTML file in a browser, or use a local server:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .

# Then visit http://localhost:8000
```

## Deployment

### GitHub Pages
This repo is configured for GitHub Pages deployment.

### Other Options
- **Vercel**: `vercel` CLI or connect repo
- **Netlify**: Drag and drop or connect repo
- **AWS S3 + CloudFront**: For enterprise deployments
- **Any static host**: Upload files via FTP/SFTP
